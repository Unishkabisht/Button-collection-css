# Button Collection — CSS Practice

A collection of 10 CSS button styles built as part of Day 4 of the Industry-Oriented Full Stack Internship.

## Buttons Included

| Button | Description |
|--------|-------------|
| Solid | Filled brown background with hover shadow |
| Outline | Transparent with brown border, fills on hover |
| Pill | Rounded pill shape with beige tone |
| Ghost | Transparent with light border, inverts on hover |
| Dark | Dark background with brown border accent |
| Beige | Light beige filled button |
| Icon | Rounded icon-style button with star |
| Underline | No border, only bottom underline |
| Wide | Extra wide solid button |
| Small | Compact small-sized button |

## Tech Used

- HTML5
- CSS3 (External stylesheet)
- Google Fonts — Inter

## Color Theme

Brown, beige, and black — `#8B4513`, `#c4956a`, `#f5e6d3`, `#1a0f0a`

## File Structure

```
buttons/
```

## How to Run

Just open `buttons.html` in any browser. No setup needed.
